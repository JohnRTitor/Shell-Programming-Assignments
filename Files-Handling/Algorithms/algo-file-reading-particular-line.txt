                Algorithm - Reading particular lines in a file

Input: Name of the file: file_name
Output: display the lines

Steps:-

1: Start
2: Read file_name
3: Read lines //line numbers to be printed
4: count ← 1
5: While line ≠ EOF do
    5.1: if count = lines then
        5.1.1: Print line
    5.2: EndIf
    5.3: count ← count+1
6: EndWhile
7: Stop
