                Algorithm - Word-Line-character-Count

Input: Name of the file (fileName)
Output: Display the number of lines, words and character in the file

Steps:

1: Start
2: Read fileName //Name of the file
3: values ← wc -l -w -m fileName //This command is used to count lines, words and characters.
4: valuesArray[] ← values. //Array assignment to extract individual values.
5: Print valuesArray[0] //Number of lines in the file
6: Print valuesArray[1] //Number of words in the file
7: Print valuesArray[2] //Number of characters in the file
8: Stop
