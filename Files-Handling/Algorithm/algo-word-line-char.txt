                Algorithm - Word-Line-Character-Count

Input: Path of the file (file_name)
Output: Display the number of lines, words and character in the file
Data Structure: One Dimensional Array

Steps:

1: Start
2: Read file_name //Name of the file
3: values_array ‚Üê file_count(file_name, LINE, WORD, CHAR) // Used to count lines, words and characters
// Equivalent to wc -l -w -m file_name command in Linux
4: Print values_array[0] //Number of lines in the file
5: Print values_array[1] //Number of words in the file
6: Print values_array[2] //Number of characters in the file
7: Stop
