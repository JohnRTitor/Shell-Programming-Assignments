                Algorithm - User Login Check

Input:- username //Name of the user
Output:- Display the login time and the duration

Steps:-

1: Start
2: If username = NULL then
    2.1: Print "Usage : type username"
    2.2: Exit
3: EndIf
4: If who | grep -wq username then
    4.1: login_time=get_login_time(username)
    4.2: duration_seconds=date() - date(login_time) //date() gives times past since epoch in seconds
    4.3: hours=duration_seconds/3600
    4.4: minutes=(duration_seconds%3600)/60
    4.5: seconds=duration_seconds%60
    4.6: echo "username is logged in since ",login_time
    4.7: echo "Duration : ",hours,minutes,seconds
5: Else
    5.1: username,"is not currently logged in"
6: EndIf
